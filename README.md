# Final Statistics Project – Boston Housing Dataset

## Assignment Overview

This project is the culmination of the statistical analysis skills learned throughout the course. Using the **Boston Housing dataset**, you will explore relationships between different variables, perform statistical analyses, and generate meaningful insights.

The assignment is provided as a **Jupyter Notebook**, containing both **markdown** and **code cells**. Clear explanations and well-formatted outputs are essential for effective communication of your results.

## Objectives

By completing this project, you will:

* Apply **descriptive statistics** to summarize and understand the dataset.
* Conduct **hypothesis testing** to make data-driven inferences.
* Perform **correlation analysis** to explore relationships between variables.
* Build **regression models** to predict outcomes and understand variable impacts.
* Present findings clearly using well-documented code and markdown explanations.

## Dataset

The project uses the **Boston Housing dataset**, which includes information about housing values in Boston suburbs along with various predictor variables such as:

* Crime rate (`CRIM`)
* Proportion of residential land zoned (`ZN`)
* Average number of rooms per dwelling (`RM`)
* Distance to employment centers (`DIS`)
* And other socioeconomic and environmental factors
